ABOUT THE PROJECT

Bienvenue. HyperText Transfer Protocol (HTTP) is one of the fundamentals of the internet. It is the mechanism that enables computers to talk to each other by defining the format in which messages are passed.
This project consisted in writing our own HTTP server. We followed the real HTTP/1.1 RFC specifications 7230-7235. This server can be tested with a real browser.
